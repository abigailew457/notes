CSS Functions Functions are reusable pieces of code which perform specific tasks. examples include: color: rgb(0, 42, 255); background: linear-gradient(90deg, blue, red); the value color is the function rgb(). Functions can not be created in CSS it comes with a bundle of premade functions like calc(), min(), max(), and clamp(). calc() handles complex math and allows you to mix units like vh, rem, and px example: :root { --header: 3rem; --footer: 40px; --main: calc(100vh - calc(var(--header) + var(--footer))); } calc() does the math 100vh - (header + footer) or 100vh - (3rem + 40px) mixing units and applying the results to main. min() acts as a boundary for the maximum allowed value. In the example width: min(150px, 100%) when there are 150px or more available then it will take up only 150px but if there are less then 150px it will take up 100% of the parents width. You can also do basic math inside a min() example: width: min(80ch, 100vw - 2rem); max() works the same as min() but in reverse. It will take the largest value in the parentheses for example width: max(100px, 4em, 50%) as long as 4em and 50% are larger then 100px max() will select the larger of the two. If however they are smaller then it will default to 100px. clamp() makes elements fluid and responsive. It takes three values the smallest, the ideal, and the largest values. In the example font-size: clamp(320px, 80vw, 60rem) 320px would be the smallest acceptable font-size, 60rem would be the largest, and 80vw would be the ideal.
